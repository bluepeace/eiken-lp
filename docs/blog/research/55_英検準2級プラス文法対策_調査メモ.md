## 1. 重要な整理（公式に基づく）

- 英検準2級プラスの一次試験に、**「文法」という独立した科目や大問名はない**（公式「準2級プラスの試験内容」の表参照）。

### リーディング（31問）で文法が効く部分

- **短文の語句空所補充（17問）**…文脈に合う**適切な語句**（公式）。問題文の種類は**短文会話文**。準2級の「短文15＋会話文5」の**会話文独立大問はなく**、短文・会話の空所がこの17問に**統合**される形。
- **長文の語句空所補充（6問）**…**説明文**のパッセージの空所に語句を補う（公式）。準2級の2問から**6問に増加**（2級と同数）。論理のつなぎ・述語・語彙が文法と一体で問われる。
- **長文の内容一致選択（8問）**…**Eメール・説明文**（公式）。読解中心だが、時制・接続・代名詞の指示など文法理解が支えになる。

### 一次試験の時間（公式）

- リーディング・ライティングが**あわせて85分**。時間は区分されず受験者が配分。

### ライティング（記述）と文法

- **英文要約（1題）**…文章の内容を英語で要約する（公式）。**自分の文**で書くため、**文法・語法の正しさ**がそのまま採点に効く。
- **英作文（1題）**…質問に対する意見を英語で論述する（公式）。同上。

※準2級の「Eメール返信」に相当する課題は、準2級プラスの公式表では**ライティングに含まれない**（要約＋英作文のみ）。記事で準2級との違いとして明記する。

### レベル（公式）

- **目安**：**高校上級程度**（準2級プラスの試験内容ページ）。

### 二次試験（参考）

### 新設級について

| 分野 | 内容の例 |

## 4. 大問別：文法の見方・対策（リーディングメモと整合）

### 大問1（短文・会話の語句空所・17問）

- 会話が**短文大問に含まれる**ため、**定型応答**と**高校上級の文法**の両方を準備。消去法＋スピード。

### 大問2（長文の語句空所・6問）

- 問数が多いので、**段落ごとの大意**→**各空所の論理（逆接・追加・例示・結論）**の順で解く練習。準2級の2問対策を**ボリュームアップ**して適用。

### 大問3（長文の内容一致・8問）

- 設問先読み・キーワード拾い。接続詞で論理を追う。

### ライティング

- **要約**：原文の文をコピーせず、**短く正しい文**にする。一人称・時制をそろえる。
- **英作文**：段落構成＋一文ごとのチェック。

## 6. 記事用・例題（オリジナル例題／本番ではない）

### 大問1風（短文・語句空所）

**例A（such … that）**  
It was (　　) a difficult test that many students failed.　1. so　2. such　3. very　4. too  
**答え**：2. such  
**解説**：**such + a/an + 形容詞 + 名詞 + that** の型。**so** は **so difficult a test** のように語順が変わるため、空所の位置では **such**。

**例B（wish + 過去完了）**  
I wish I (　　) more time to prepare for the presentation.　1. have　2. had　3. had had　4. will have  
**答え**：3. had had  
**解説**：「（当時）もっと時間があればよかった」という**過去の事実に対する後悔**の wish で **had had** が典型（教材レベルによるが高校上級で扱われやすい）。

### 大問1風・会話のイメージ（短文会話文）

A: I'm afraid we're completely sold out.　B: (　　)  
1. That's too bad.　2. With pleasure.　3. Here you are.　4. Long time no see.  
**答え**：1. That's too bad.  
**解説**：在庫なしの知らせに対する**自然な応答**。

### 大問2風（長文の語句空所・論理）

本文イメージ：Some people support the plan. (　　), others worry about the cost.  
**1. Moreover　2. However　3. For example　4. Therefore**  
**答え**：2. However  
**解説**：**support** と **worry** の**対比**なので逆接の **However**。

### ライティング（要約・文法の心がけ）

- 要約では **The article says that ～** など**定型の導入**＋**現在形・過去形の統一**を意識する（例文レベルの話）。
