## 1. 重要な整理（公式に基づく）

- 英検4級の一次試験に、**「文法」という独立した科目や大問名はない**（公式「4級の試験内容」の表参照）。
- **文法・語形・語順**は、主に**リーディング**の次の形式のうち、**大問1～3**で総合的に問われる。
  1. **短文の語句空所補充（15問）**…文脈に合う**適切な語句**（公式）。単語・熟語に加え、**時制・不定詞・比較・接続詞・助動詞**など、中1～中2で扱う文法が選択肢・空所に現れる典型パターンが多い。
  2. **会話文の文空所補充（5問）**…会話の流れに合う**文や語句**（公式）。**依頼・提案・誘いへの応答**など、文法的に正しい文の選び方が問われる。
  3. **日本文付き短文の語句整序（5問）**…**語順**が文法理解の中心。5級より文が長くなり、**接続詞句・不定詞・動名詞**などが絡むことがある（教材・過去問の一般論）。
- **大問4（長文の内容一致選択・10問）**は、主に**読解・情報の拾い読み**が中心。空所補充ではないが、**文のつながり・時制・代名詞の指示**を読み取る力は文法の理解に支えられる。

- よって記事では「英検4級の文法問題＝大問1～3で文法面が効いてくる部分」と定義し、大問4は**補足**として触れる。

## 5. 大問別：文法の見方・対策（リーディングメモと整合）

### 大問1（語句空所）

- **時制**（いつ起きたか）、**主語と動詞の一致**、**不定詞/動名詞の型**、**比較級**などを、空所の前後から推定する。
- 意味が通っても**形が合わない**選択肢を先に消す。

### 大問2（会話の空所）

- **誘い・提案・お願い**に対する**定番の返答**を、文法とセットで覚える。
- 会話全体の流れと、**文法的整合**の両方でチェック。

### 大問3（整序）

- 日本文を読み、**主語と動詞**を先に置く。**接続詞（When ～, ...）**や**不定詞句**の塊を意識する。
- 4級は文が長めになりやすいので、**塊（句）**で並べ替える練習が有効。

### 大問4（長文・補足）

- 設問先読み・キーワード拾いが中心。**代名詞のさし示し**や**時制の一致**で迷わないよう、基礎文法は大問1～3と共通で固める。

## 7. 記事用・例題（オリジナル例題／本番ではない）

### 大問1風（語句空所）

**例A（過去形）**  
I (　　) a movie yesterday.　1. see　2. saw　3. seeing　4. sees  
**答え**：2. saw  
**解説**：yesterday は過去の合図。**see** の過去形は **saw**。

**例B（不定詞）**  
She wants (　　) English in Canada.　1. study　2. to study　3. studying　4. studied  
**答え**：2. to study  
**解説**：**want** のあとは **to + 動詞の原形**（want to study）。

**例C（比較級）**  
This river is (　　) than that one.　1. long　2. longer　3. longest　4. the longest  
**答え**：2. longer  
**解説**：**than** があるので**比較級** **longer**。

### 大問2風（会話の空所）

A: Would you like some juice?　B: (　　)  
1. Yes, please.　2. I'm ten.　3. It's sunny.　4. See you tomorrow.  
**答え**：1. Yes, please.  
**解説**：**Would you like ～?** への丁寧な肯定の返答として **Yes, please.** が自然。

### 大問3風（整序）

日本文：彼女は泳ぐことが好きです。  
語句：**swimming / She / likes / .**  
**答え**：She likes swimming.  
**解説**：主語 **She**、動詞 **likes**、目的語として動名詞 **swimming**。
