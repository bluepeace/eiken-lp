# 調査メモ：英検 合格率（各級の合格率・公表の歴史）

## 調査日・目的

- キーワード：英検 合格率 各級
- 合格率が現在公表されていないこと、過去の公表状況、参考になる「目安」の扱いを整理する

---

## 1. 現在の公式の公表状況

### 日本英語検定協会「受験の状況」

- **URL**：https://www.eiken.or.jp/eiken/about/situation/
- **公表している内容**：**志願者数**のみ（年度別・区分別：小学校以下、中学・高等学校、大学、その他、合計）
- **公表していない内容**：**合格者数・合格率**は掲載されていない
- 実用英語技能検定・英検IBA・英検Jr. の**合算志願者数**として表記

### 合格率の公表廃止時期

- **2015年度以降**（または**2016年度以降**としている資料あり）、級別の**合格者数・合格率は公式には公表されていない**。
- 過去には**1990年代～2014年頃**まで、級別の志願者数・合格者数・合格率（またはそれに類する統計）が公表されていたとする情報が複数ある（第三者サイトのグラフ等：例「英検級別受験者数と合格率の年推移（1993-2020）」）。
- 協会が「合格率を公表しなくなった」理由の公式説明は、検索範囲では見つかっていない。志願者数は「英検」ブランド合算として公表し、合格率は出していない。

## 2. 過去データから言及される「おおよその合格率」（非公式の目安）

※いずれも**公式の現行データではなく**、過去の公表値や各種解説を参考にした**目安**。

| 級     | 合格率の目安（参考） | 備考 |
|--------|----------------------|------|
| 5級    | 約80%前後            | 過去の公表値で 81% 台等の記述あり |
| 4級    | 約70%前後            | 過去の公表値で 69～70% 台等 |
| 3級    | 約50%台              | 二次（面接）が加わるため低下 |
| 準2級  | 約35%前後～          | 級が上がるほど低下傾向 |
| 2級    | 約25～30%前後        |  |
| 準1級  | 約15%前後            |  |
| 1級    | 約10%前後            | 最難関 |

- 準2級プラスは2025年度新設のため、公表された合格率は存在しない。準2級と2級の中間程度と想定する説明が多い。

## 3. 記事化のポイント

- **「現在、英検協会は級別の合格率を公表していない」**ことをはっきり書く。
- **「かつては公表されていたが、2015年頃を境に公表されなくなった」**と簡潔に触れる（正確な年度は「2015年度以降」でよい）。
- 現在公式で分かるのは**「受験の状況」の志願者数**だけであることを伝える。
- 「塾・教材などの記事では『合格率の目安』が載ることがあるが、それは**過去のデータや経験に基づく参考値**であり、**公式の現行の合格率ではない**」と注意を促す。
- 参考として「5級・4級は合格率が高め、3級から面接が加わりおおよそ50%台、2級以上はさらに低い」といった**傾向の説明**に留め、数値は「目安」「参考」と断る。
- 保護者向けに「合格率より、お子さんの実力に合った級で挑戦し、過去問で対策することが大切」と締めてもよい。
