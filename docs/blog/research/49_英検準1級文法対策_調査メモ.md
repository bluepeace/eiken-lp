## 5. 大問別：文法の見方・対策（リーディングメモ33と整合）

### 大問1（短文・会話の語句空所・18問）

- **語彙と語法の知識**が効く。難問に固執せずテンポよく消去し、ライティングの時間を確保。

### 大問2（長文の語句空所・6問）

- **タイトル**で論旨を想定。評論文では**主張→理由→反論→結論**の流れと接続語の対応を意識。

### 大問3（長文の内容一致・7問）

- 設問先読み・根拠の特定。**否定語・限定語（only, merely）**のスコープを文法として確認。

### ライティング

- 要約：**一人称を避ける**、時制の統一、パラフレーズ。
- 英作文：**論旨の明確化**、接続詞、文法ミスの最終チェック。

## 7. 記事用・例題（オリジナル例題／本番ではない）

### 大問1風（短文の語句空所）

**例A（It is essential that + 原形）**  
It is essential that every applicant (　　) the form by Friday.　1. submit　2. submits　3. submitted　4. submitting  
**答え**：1. submit  
**解説**：**It is essential that ～** の that 節では**仮定法現在（原形）**が選ばれやすい（フォーマル文体）。

**例B（Were it not for …）**  
(　　) for your support, we could not have finished the project.　1. If it is not　2. Were it not　3. Unless it is not　4. Had it not  
**答え**：2. Were it not  
**解説**：**Were it not for ～**（～がなければ）は仮定法の倒置の定番形。

**例C（接続詞・譲歩）**  
(　　) the high cost, many consumers still choose organic food.　1. Because　2. Despite　3. Although　4. However  
**答え**：2. Despite  
**解説**：空所のあとが**名詞句 the high cost**なので前置詞の **Despite**（Although は節が続く）。

### 大問2風（長文の語句空所・論理）

本文イメージ：The policy reduced unemployment. (　　), it widened income inequality.  
**1. Therefore　2. Similarly　3. Conversely　4. First**  
**答え**：3. Conversely  
**解説**：一見ポジティブな結果のあとにネガティブな帰結で**対比・逆方向**なので **Conversely**（文脈により However 等もあり得るが選択肢で区別）。

### 大問1風・会話のイメージ

A: The results won't be available until next week.　B: (　　)  
1. I see. We'll have to wait, then.　2. Congratulations!　3. Never mind.　4. Here you are.  
**答え**：1. I see. We'll have to wait, then.  
**解説**：**遅延・待機**の文脈に合う応答。
