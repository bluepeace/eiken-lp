## 1. 重要な整理（公式に基づく）

- 英検1級の一次試験に、**「文法」という独立した科目や大問名はない**（公式「1級の試験内容」の表参照）。

### リーディング（35問）で文法が効く部分

- **短文の語句空所補充（22問）**…文脈に合う**適切な語句**（公式）。問題文の種類は**短文会話文**。**大学上級程度**の**語彙・語法・文法**が集中的に問われ、1級対策では**比重が最も大きい**大問（教材・受験体験の一般論）。
- **長文の語句空所補充（6問）**…**説明文・評論文など**のパッセージの空所に語句を補う（公式）。学術的・論説的な**論理のつなぎ**と**コロケーション**が文法と一体で問われる。
- **長文の内容一致選択（7問）**…パッセージの内容に関する質問に答える（公式）。読解中心だが、**否定・モダリティ（may, must, unlikely 等）・照応**を正確に読む力は文法・語法の理解に支えられる。

### 一次試験の時間（公式）

- リーディング・ライティングが**あわせて100分**。時間は区分されず受験者が配分。

### ライティング（記述）と文法（公式）

- **英文要約（1題）**…文章の内容を英語で要約する。
- **英作文（1題）**…**指定されたトピックについての意見**を英語で論述する。

### 準1級との違い（公式・プロジェクト内リーディングメモ34と整合）

- 短文空所が**22問**（準1級は18問）で、**語彙・語法問題のボリューム**が最大。長文語句6・長文一致7は問数は準1級と同じ。
- 一次**100分**（準1級は90分）。話題は同系統だが**専門性・抽象度・語数**がさらに上がるイメージ。

### 二次試験（参考）

- 面接委員**2人**。評価に**文法**（公式表では「文法、発音の正確さなど」）。**スピーチ（2分）**と**Q&A**で複雑な文・接続を要する。

## 5. 大問別：文法の見方・対策（リーディングメモ34と整合）

### 大問1（短文・会話の語句空所・22問）

- **語彙と語法の暗記量**が合格の鍵。消去法＋テンポ。分からない問はマークして後回し。

### 大問2（長文の語句空所・6問）

- タイトル・段落大意→論理関係。評論文の**主張―根拠―反論**の型と接続語の対応。

### 大問3（長文の内容一致・7問）

- 設問先読み・根拠特定。**推論と事実**の区別、**言い換え**の正否を文法的に検証。

### ライティング

- 要約：**中立の文体**、時制、パラフレーズ。
- 英作文：**トピックに即した論旨**、接続詞、文法エラーの最終チェック。

## 7. 記事用・例題（オリジナル例題／本番ではない）

### 大問1風（短文の語句空所）

**例A（否定語句始まりの倒置）**  
Under no circumstances (　　) the confidential data with outsiders.　1. you should share　2. should you share　3. you will share　4. you shared  
**答え**：2. should you share  
**解説**：**Under no circumstances** が文頭なので**倒置**（助動詞＋主語＋動詞）。

**例B（provided that）**  
Employees may work remotely (　　) their supervisor approves it in advance.　1. unless　2. provided that　3. although　4. however  
**答え**：2. provided that  
**解説**：「**条件が満たされれば許可**」の **provided that**（as long as 等も文脈による）。

**例C（コロケーション）**  
The government decided to (　　) strict sanctions on the region.　1. impose　2. compose　3. propose　4. oppose  
**答え**：1. impose  
**解説**：**impose sanctions on ～** は定番のコロケーション。

### 大問2風（長文の語句空所）

本文イメージ：The data do not prove causation. (　　), the findings are consistent with earlier studies.  
**1. Therefore　2. Nevertheless　3. By the same token　4. Conversely**  
**答え**：2. Nevertheless  
**解説**：前が限定（証明しない）、後が肯定（整合する）で**譲歩・それでも**のニュアンスなら **Nevertheless**（文脈により Still 等）。

### 大問1風・会話のイメージ

A: I'm not sure this proposal is feasible in the current budget.　B: (　　)  
1. I take your point. We need to revisit the numbers.　2. That's wonderful news.　3. With pleasure.　4. Never mind.  
**答え**：1. I take your point. We need to revisit the numbers.  
**解説**：懐疑的な意見に**建設的に応じる**フォーマルな会話。
