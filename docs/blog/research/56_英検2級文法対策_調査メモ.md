- **長文の語句空所補充（6問）**…**説明文**のパッセージの空所に語句を補う（公式）。論理のつなぎ（ディスコースマーカー）や述語・語彙が文法と一体で問われる。
- **長文の内容一致選択（8問）**…**Eメール・説明文**（公式）。読解中心だが、時制・接続・代名詞の指示など文法理解が支えになる。

## 4. 2級で押さえておきたい文法項目（教材・問題集の一般論）

※公式の出題リストではない。準2級プラス（高校上級）の土台に、**高校卒業付近で頻出**しやすい項目を列挙。

| 分野 | 内容の例 |
|------|----------|
| 準2級プラスまでの土台 | such/so、wish、倒置の入り口、長文の論理マーカー、コロケーション |
| 助動詞＋受動 | should be done, must have been ～ など |
| 関係詞の複合 | reason why、先行詞と制限・非制限、what 節 |
| 仮定法過去完了・混成 | If I had known, … would have ～ など（教材範囲） |
| 強調・倒置 | It is ～ that ～、否定語始まりの倒置の選択肢区別 |
| 分詞・独立分詞構文 | 意味上の主語、with + O + 過去分詞 など |
| 名詞節・形容詞のパターン | whether, the fact that, be likely to など |
| 長文語句空所（6問） | Nevertheless, Consequently, On the other hand, That is など**やや硬い論理表現** |
| ライティング | 要約の**名詞化・受動**、作文の**接続詞・段落構成**、文法ミスの自己チェック |

## 5. 大問別：文法の見方・対策（リーディングメモ32と整合・補足）

### 大問1（短文・会話の語句空所・17問）

- 消去法＋スピード。語彙と文法の**セット**で暗記。

### 大問2（長文の語句空所・6問）

- タイトル・段落大意→各空所の論理。**前後2文**まで読む（プロジェクト内リーディングメモのコツ）。

### 大問3（長文の内容一致・8問）

- 設問先読み・キーワードスキャン。選択肢の**述語・限定語**の差を意識。

- **英作文**：トピックに沿った**論旨＋根拠＋結論**、文法チェック。

## 7. 記事用・例題（オリジナル例題／本番ではない）

### 大問1風（短文の語句空所）

**例A（受動＋助動詞）**  
The problem should (　　) immediately.　1. solve　2. be solved　3. have solved　4. solving  
**答え**：2. be solved  
**解説**：problem は「解かれる」ので受動。**should** のあとは原形、**be solved**。

**例B（reason why）**  
The reason (　　) he left the company is still unknown.　1. why　2. which　3. what　4. how  
**答え**：1. why  
**解説**：**The reason why ～** で「～した理由」。

**例C（仮定法過去完了の結果）**  
If I had studied harder, I (　　) the exam.　1. pass　2. passed　3. would pass　4. would have passed  
**答え**：4. would have passed  
**解説**：**If I had p.p.** に対する主節は **would have + 過去分詞** が典型。

### 大問2風（長文の語句空所・論理）

本文イメージ：The technology saves energy. (　　), it is too expensive for small businesses.  
**1. Moreover　2. Nevertheless　3. For instance　4. First**  
**答え**：2. Nevertheless  
**解説**：前が利点・後が問題で**譲歩・対比**なので **Nevertheless**。

### 大問1風・会話のイメージ

A: Do you think we should leave now?　B: (　　) The traffic will get worse.  
**1. I'm afraid so.　2. Never mind.　3. With pleasure.　4. My pleasure.**  
**答え**：1. I'm afraid so.  
**解説**：後続の理由と合わせ、**そう思う（残念ながら）** の **I'm afraid so.**
