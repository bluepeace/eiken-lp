## 1. 重要な整理（公式に基づく）

- 英検準2級の一次試験に、**「文法」という独立した科目や大問名はない**（公式「準2級の試験内容」の表参照）。

### リーディング（29問）で文法が効く部分

- **短文の語句空所補充（15問）**…文脈に合う**適切な語句**（公式）。単語・熟語に加え、**高校中級程度**の文法（分詞・関係詞・接続詞・仮定法の入り口・不定詞構文など）が選択肢・空所に現れる。
- **会話文の空所補充（5問）**…空所に適切な**文や語句**（公式）。やや丁寧・形式的な会話表現と文法の整合が問われる。
- **長文の語句空所補充（2問）**…物語文・説明文のパッセージの空所に語句を補う（公式）。**3級にはない形式**。段落の**論理のつながり**（However, Therefore, For example など）や、文脈に合う**述語・接続表現**が問われやすい（教材・過去問の一般論）。
- **長文の内容一致選択（7問）**…読解が中心。本文の**時制・代名詞・論理接続**を読み取る力は文法理解に支えられる。

### 一次試験の時間（プロジェクト内リーディングメモと整合）

## 4. 準2級で押さえておきたい文法項目（教材・問題集の一般論として整理）

※「この項目が必ず出る」という公式リストではない。3級までの土台に、**高校基礎～中級で頻出しやすい追加**を列挙する。

| 分野 | 内容の例 |
|------|----------|
| 3級までの土台 | 完了形・受動態・関係代名詞・間接疑問・不定詞/動名詞など |
| 分詞 | 現在分詞・過去分詞、分詞構文の読み方（意味上の主語との関係） |
| 関係詞の拡張 | 前置詞＋関係代名詞、what 節の初歩など |
| 仮定法・条件 | If I were ～, I would ～ などの基本、wish の初歩 |
| 接続詞・副詞 | although, unless, so that, in order that、however, therefore など |
| 比較・最上級の複合 | the more ～, the more ～ の入り口など（教材による） |
| 強調構文の入り口 | It is ～ that ～（認識レベル） |
| 長文語句空所向け | **ディスコースマーカー**（However, Therefore, For example, In addition, First など）の意味と前後の論理 |
| ライティング | 段落内の時制統一、接続詞で理由・対比・結論を示す |

## 5. 大問別：文法の見方・対策（リーディングメモと整合）

### 大問1（短文の語句空所）

- 文全体の**論理・時制**、**分詞か謂語動詞か**、**接続詞の意味**で消去する。
- 80分のなかでライティングの時間を確保し、**スピードと消去法**を両立。

### 大問2（会話の空所）

- 丁寧な依頼・提案への**定型応答**を文法とセットで整理。

### 大問3（長文の語句空所・2問）

- **段落の前後**で「逆接・因果・例示・順序」のどれかを判断し、つなぎ語句を選ぶ。
- タイトル・各段落の最初の文で**大意**を掴んでから空所へ。

### 大問4（長文の内容一致）

- 設問先読み・キーワード拾い。接続詞で論理展開を追うと文法知識が活きる。

### ライティング

- メール・意見文の**型**＋一文ごとの**主語・動詞・時制**チェック。

## 7. 記事用・例題（オリジナル例題／本番ではない）

### 大問1風（短文の語句空所）

**例A（仮定法過去の基本）**  
If I (　　) you, I would talk to the teacher first.　1. am　2. was　3. were　4. be  
**答え**：3. were  
**解説**：**If I were you**（もし私があなたなら）は定番の仮定法過去の型。

**例B（分詞）**  
(　　) tired, she went to bed early.　1. Felt　2. Feeling　3. Feels　4. To feel  
**答え**：2. Feeling  
**解説**：主語 **she** と**能動の意味**なので、分詞の **Feeling** で「疲れを感じながら」。

**例C（目的）**  
He studied hard (　　) he could pass the exam.　1. although　2. unless　3. so that　4. however  
**答え**：3. so that  
**解説**：「合格できる**ように**」という**目的**なので **so that**。

### 大問3風（長文の語句空所・論理）

短い本文イメージ：Many people like online shopping. (　　), some still prefer going to stores.  
**1. However　2. Therefore　3. For example　4. First**  
**答え**：1. However  
**解説**：前後が**対比・逆接**なので **However**。

### 大問2風（会話の空所）

A: Would you mind closing the window?　B: (　　)  
1. Not at all.　2. Never mind.　3. Here you are.　4. With pleasure.（※選択肢は「Not at all」が無難。With pleasure は文脈によって可だが、Would you mind は Not at all が定番解説されやすい）  
**答え**：1. Not at all.  
**解説**：**Would you mind ～ing?** に対する**許可・了承**の返答として **Not at all.** が教材でよく扱われる。
