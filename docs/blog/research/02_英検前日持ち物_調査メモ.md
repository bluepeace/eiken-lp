# 調査メモ：英検 前日確認・持ち物

## 調査日・目的

- キーワード：英検 前日確認 持ち物
- 受験者（お子さん）の保護者向けに、前日・当日のチェックリストを整理

---

## 一次ソース・信頼できる情報

- 日本英語検定協会公式サイトの受験案内
- 各塾・予備校の「英検 持ち物」まとめ（複数サイトで共通する内容を採用）

---

## 必須の持ち物（従来型・1級〜5級）

### 全受験者共通

| もの | 備考 |
|------|------|
| **受験票** | **紙で印刷したもの**。スマホ画面のみでは不可。 |
| **身分証明書** | 運転免許証、パスポート、健康保険証、学生証など。**2026年度第1回から**1級〜3級は**顔写真付き身分証明書の原本**が必須（リセマム報道）。 |
| **腕時計** | アナログ・デジタル可。**スマートウォッチは不可**。試験中の時間確認用。 |

### 1級〜3級（一次＋二次で必要な場合）

| もの | 備考 |
|------|------|
| 一次受験票兼本人確認票 | 案内に従って持参 |
| **顔写真** | **縦3cm×横2.4cm**（運転免許証と同じサイズ）。正面・6ヶ月以内・帽子なし・髪が顔にかからないこと。 |

### 筆記用具（従来型の一次試験）

| もの | 備考 |
|------|------|
| 黒鉛筆 or シャープペン | **HB または B**。Hや2Bは不可の案内が多い。 |
| 消しゴム | 無地が無難 |

### 会場による指定

| もの | 備考 |
|------|------|
| 上履き・靴袋 | **受験票に記載がある場合のみ**持参 |

---

## あると便利なもの

- **クリアファイル** … 受験票や証明書の保護・雨対策
- **上着** … 冷房が効いている会場が多いため
- **ハンカチ・ティッシュ**
- **飲み物** … 休憩時間のみ可。試験中は飲めない
- **単語帳や軽い復習用のもの** … 待ち時間用（会場の指示に従う）

---

## 持込み禁止（共通で注意）

- スマートフォン（電源オフでも不可の会場が多い）
- スマートウォッチ
- 電子辞書
- イヤホン・ヘッドホン
- 参考書・問題集（試験中）
- 帽子・フード付き衣類（本人確認の妨げになるため禁止の会場あり）

---

## 前日・当日のチェックのポイント

1. **受験票**を印刷し、必要なら顔写真を貼る（1級〜3級）。
2. **身分証明書**を用意（1級〜3級は顔写真付き）。
3. **腕時計**はスマートウォッチでないもの。
4. **筆記用具**は HB/B の鉛筆 or シャープ＋消しゴム。
5. 会場の**到着目安**は開始**30〜40分前**が一般的。
6. 顔写真を忘れた場合は、会場近くの証明写真機・コンビニで撮影する案内あり。

---

## 記事化のポイント

- 「前日の夜に一緒にリュックに入れておこう」など、親子でできる具体的な行動に落とす
- スマホ・スマートウォッチは「持っていかない」「預ける」をはっきり書く
- 受験票は「印刷必須」「スマホ画面だけではダメ」を強調
- 2026年度から1〜3級で顔写真付き身分証必須になった点に触れると親切

## 補足（増量用）

- 受験票を紙で印刷する理由：会場での本人確認のしやすさ・不正防止のため。スマホ画面のみは不可としている会場が一般的。
- 忘れ物をした場合：顔写真は会場近くの証明写真機・コンビニで撮影可能。受験票・身分証を忘れた場合は会場に問い合わせを。
