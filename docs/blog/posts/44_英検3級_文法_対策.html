英検3級の対策では、「文法はどこで点になるの？」と気になる保護者の方も多いです。
結論から言うと、3級にも「文法」という名前の独立したパートはありません。
ただし、リーディングの大問1・2では選択式で文法が問われ、ライティング（Eメール・英作文）では自分で書く英文の正しさとして文法がそのまま評価されます。
この記事では、どの問題で文法が効いてくるか、4級から足すとよいポイント、家庭でできる勉強法を、わかりやすくまとめました。

英検3級で「文法」が出る場所（公式の形式）

英検公式の「3級の試験内容」では、一次試験はリーディング・ライティングがあわせて65分、続けてリスニング（約25分）です。
リーディングとライティングの時間は区切られていないため、文法対策とあわせて時間配分も意識しておくと安心です。

大問1：短文の語句空所補充（15問）

短い英文の（　）に、文脈に合う語句を選ぶ問題です。
公式では「文脈に合う適切な語句」とあり、単語・熟語に加え、時制・受動態・関係代名詞・不定詞/動名詞・接続詞など、文法まわりが頻出します。
多くの教材では中学卒業程度の範囲が目安と説明されています。

大問2：会話文の空所補充（5問）

会話の（　）に、適切な文や語句を選びます。
「文法的に正しいか」に加えて、「直前の発話に自然につながるか」も大事です。
依頼・提案・断り・同意などの定型表現が、文法とセットで問われます。

大問3：長文の内容一致選択（10問）※文法との関係

掲示・案内、Eメール（手紙文）、説明文を読んで、内容に関する質問に答えます。
設問の中心は読解ですが、本文の時制・接続詞・代名詞のさし示しを読み取る力は、基礎文法の理解に支えられます。

ライティング：Eメールと英作文（各1題）

3級の大きな特徴は、自分で英文を書くパートがあることです。
返信メールを英文で書く問題と、質問に対する意見を英語で論述する問題があり、採点では文法の正しさや語法が直接効きます。
「選択式の文法ができる」と「記述でミスなく書ける」は別なので、ライティング用の型・自己チェックも必要です。

4級とのちがい：語句整序（並べ替え）がない

4級・5級のリーディングにある日本文付きの語句整序は、3級の公式の試験内容にはありません。
語順のルールは、長文を読むときやライティングで書くときに、引き続き重要です。

レベル感（公式の目安）

3級は公式の目安で中学卒業程度です。
4級の「中学中級」の次の段階として、文の長さ・語彙・文法の幅が広がるイメージで捉えるとわかりやすいです。

3級で押さえておきたい文法のポイント

以下は、3級対策の参考書や問題集でよく登場する項目です（公式の出題リストではありません）。
4級までの土台に加え、次をチェックするとよいです。

  • 現在完了形：have/has + 過去分詞（経験・結果など、文脈との対応）。
  • 受動態：be + 過去分詞。いつの話か（時制）とセットで。
  • 関係代名詞：who, which, that などと先行詞の対応。
  • 間接疑問：I don't know ～ などのあとで主語＋動詞の語順になる、など。
  • 不定詞・動名詞：意味の違いがつきやすい動詞（stop to do / stop doing など、教材で扱う範囲）。
  • 接続詞・文のつながり：because, although, however などで理由・対比を読み書きする。
  • ライティング：主語と動詞の一致、段落のなかで時制をそろえる、First / Also / In conclusion などでつなぐ。

大問別の対策のコツ（文法の見方）

大問1（語句空所）

空所の前後から、「能動か受動か」「いつの話か」「先行詞は人か物か」を推測します。
65分のなかでライティングの時間も必要なので、消去法で悩みすぎない練習も大切です。

大問2（会話の空所）

直前の一文が質問か依頼かを確認し、文法的にも会話としても自然な応答を選びます。
過去問で出た定型をノートにまとめ、声に出して言い換えると定着しやすいです。

大問3（長文）

設問先読み・キーワード拾いが中心ですが、本文では接続詞の前後に理由や逆接の情報が集まりやすいので、長文読解の文法として意識するとよいです。

ライティング（記述）

Eメールは挨拶・本文・結びの型を覚え、英作文は意見＋理由＋結論の骨組みを日本語で先に決めてから英語にします。
書き終えたあと、各文の主語と動詞の形だけでも見直す習慣をつけると減点を防ぎやすくなります。

日々の勉強法

中学卒業程度の文法を問題集で単元ごとに進め、英検3級の過去問で「英検の出し方」に慣れます。
65分まるごとでリーディング＋ライティングを解く練習をすると、文法だけでなく時間配分も身につきます。
間違えた空所は、「受動態」「関係代名詞」など項目名をメモして復習リストにするとよいです。

例題で形式をイメージする（オリジナル例）

※ 本番の予想問題ではなく、形式の理解用の簡単な例です。

大問1風・例1（受動態）

The window (　　) by Tom yesterday.　1. broke　2. was broken　3. breaks　4. is broken
答え：2. was broken
窓が「壊された」ので受動態。yesterday から過去の受動、was broken が合います。

大問1風・例2（関係代名詞）

This is the book (　　) I read last week.　1. who　2. which　3. whose　4. whom
答え：2. which
先行詞が物（book）なので which を選びます（選択肢に that があれば文脈によって同様、という説明も教材でありがちです）。

大問1風・例3（現在完了）

I (　　) this song many times.　1. hear　2. heard　3. have heard　4. am hearing
答え：3. have heard
many times は経験を表す現在完了と結びつきやすい合図です。

大問2風・例（依頼への返答）

A: Could you help me with my homework?　B: (　　)
1. Sure, no problem.　2. I'm fine, thanks.　3. It's Monday.　4. See you later.
答え：1. Sure, no problem.
Could you ～? への肯定的な返答として自然です。

ライティングの自己チェック（イメージ）

記述問題は例題の選択肢がありませんが、家で練習するときは次を親子で確認するとよいです。
・一文に動詞は1つ（従属接続詞でつないでいないか）
・主語が三人称単数なら動詞に -s がつくか
・過去の出来事なら時制がそろっているか

二次試験と文法（参考）

3級の二次試験（面接形式のスピーキング）では、公式の試験内容に文法・語法が評価の観点のひとつとして示されています。
一次試験の文法対策で身につけた基本的な文の型は、面接での応答にもそのまま活きます。

何を使って勉強するか

文法は、中学の教科書・ワークで土台を作り、英検の過去問でリーディングの形とライティングの時間配分に慣れるのが定番です。
英検対策アプリ「AiKen（アイケン）」では、英検で出やすい単語や本番に近い類似問題をたくさん用意しています。
単語テストで文法とセットになった語句を確認したり、間違えた問題を記録したり、音読機能で例文を声に出したりできるので、語句空所の反復に活用できます。
