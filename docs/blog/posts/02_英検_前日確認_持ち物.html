英検の試験日が近づいてくると、「当日何を持っていけばいいんだっけ？」と不安になりますよね。
前日の夜に、お子さんと一緒にリュックに入れておけるように、持ち物をまとめました。
このリストを印刷して、チェックしながら準備すると忘れ物が減ります。

これだけは絶対に忘れずに（必須の持ち物）

全員共通で必要なもの

  • 受験票　…　紙で印刷したものを持っていきます。
    スマホの画面だけでは受験できないので、必ず印刷してください。
    会場では本人確認に使うため、紙の受験票が必要とされています。
  • 身分証明書　…　運転免許証、パスポート、健康保険証、学生証などです。
    2026年度の第1回から、1級・2級・準1級・3級の受験では顔写真のついた身分証明書の原本が必要になります。
    お子さんの場合は学生証（写真付き）や、パスポートなどで確認してください。
    小学生で写真付きの身分証がない場合は、事前にパスポートを取得するか、受験案内で認められている方法を確認しておきましょう。
  • 腕時計　…　試験中の時間確認用です。
    スマートウォッチは使えないので、普通の腕時計（アナログでもデジタルでも可）を用意してあげてください。
    会場の時計が見づらい席になることもあるので、自分で時間が分かると安心です。

1級・2級・準1級・3級の場合、さらにこれ

  • 顔写真　…　縦3cm×横2.4cm（運転免許証と同じサイズ）です。
    正面を向いた、6ヶ月以内に撮ったものを使います。
    帽子をかぶっていない、髪が顔にかかっていないようにしてください。
    忘れた場合は、会場近くの証明写真機やコンビニで撮ることもできます。
    前日までに貼り忘れがないか、受験票と一緒に確認しておきましょう。

筆記用具（一次試験で使うもの）

  • 黒い鉛筆またはシャープペン　…　HBかBの濃さのものです。
    Hや2Bは不可としている会場が多いので、必ずHBかBを用意してください。
    予備も1本あると安心です。
  • 消しゴム　…　普段使っているもので大丈夫です。
    マークを塗り直すときに使いやすい、きれいに消えるものを。

会場によっては

  • 上履き・靴袋　…　受験票に「上履き持参」などと書いてある場合だけ持っていきます。
    書いていなければ不要です。
    会場によっては土足のまま入れるところもあるので、受験票の注意書きを必ず読んでおきましょう。

あると便利なもの

  • クリアファイル　…　受験票や身分証明書が折れたり、雨で濡れたりしないように入れられます。
    小さなお子さんだとカバンの中で書類がくしゃくしゃになりがちなので、あると便利です。
  • 上着　…　会場の冷房が効いていることが多いので、1枚あると安心です。
    脱ぎ着しやすいカーディガンやパーカーがおすすめです。
  • ハンカチ、ティッシュ　…　ちょっとした汗や鼻水対策に。
  • 飲み物　…　休憩時間にだけ飲めます。
    試験中は飲めないので、フタ付きのものを。
    ペットボトルはラベルをはがして持っていくよう案内している会場もあります。
  • 単語帳や軽い復習の本　…　待ち時間用です。
    会場のルールに従って、見てよい時間帯かどうか確認してください。

持っていってはいけないもの

お子さんに「これは持っていかないで」と伝えておいてください。
持っているだけで注意されることがあるので、前日にカバンから出しておくか、保護者が預かるようにしましょう。

  • スマートフォン（電源を切っていても、持っているだけでルール違反になる会場があります。
    会場に預ける場合の案内があれば、それに従ってください。）
  • スマートウォッチ　…　時計として使えても、通信機能があるため不可の会場がほとんどです。
  • 電子辞書
  • イヤホン・ヘッドホン
  • 参考書や問題集（試験中は見られません。
    待ち時間用も会場の指示を確認してください。）
  • 帽子やフード付きの服　…　本人確認のときに邪魔になるため、禁止している会場があります。
    当日の服装も、前日に一緒に決めておくとよいです。

前日の夜にやっておきたいこと

  1. 受験票を印刷し、必要な級なら顔写真を貼る。
    スマホやPCの画面で「印刷」ボタンから、自宅のプリンターかコンビニで印刷できます。
  2. 身分証明書を用意する（1～3級は顔写真付きのもの）。
    有効期限が切れていないかも確認しましょう。
  3. 腕時計がスマートウォッチではないことを確認。
    電池切れしていないかもチェック。
  4. 筆記用具（HBまたはBの鉛筆 or シャープ＋消しゴム）を用意。
    シャープペンは芯が十分か見ておいてください。
  5. 持ち物を全部リュックやカバンに入れて、玄関に置いておく。
    朝バタバタすると忘れ物しやすいので、前日のうちにまとめておくのがおすすめです。

試験会場には、開始30分～40分前を目安に到着するようにすると余裕があります。
会場が初めての場所の場合は、前日までに経路を確認しておくといいですね。
「何時に家を出るか」まで決めておくと、当日の朝がスムーズです。
前日の夜は、お子さんがしっかり眠れるように、いつもより少し早めに休むようにしてあげてください。

当日の朝、もう一度確認

「受験票」「身分証明書」「腕時計」「鉛筆・消しゴム」の4つは、朝もう一度チェックしてから家を出ましょう。
忘れ物があると受験できない場合もあるので、前日の準備が本当に大切です。

「頑張ってね」と声をかけて、リラックスして会場に向かえるようにしてあげるのもいいと思います。
